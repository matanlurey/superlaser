  1. @nonVirtual
void resetHealthModifier()

Resets the health modifier on this target (i.e. sets it to 0).

This is a convenience method to reset the health modifier to its default value. It is recommended to use this method instead of setting the health modifier directly to 0 to make interactions more explicit.

For example, when recalculating the health modifier based on active upgrades, it is recommended to call this method, and then iterate over the active upgrades to apply their health modifiers.

Implementation

@nonVirtual

  healthModifier = 0;
}
