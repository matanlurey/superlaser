engine library

A gameplay engine for the Star Wars: Unlimited Trading Card Game.

This library provides gameplay logic and data structures. It is intended to be used when building anything like a simulator, a game client, or a game visualizer.

Usage

import 'package:unlimited/engine.dart';

Scope

This library provides static type checking and some low-level correctness checks. For example, the Base class can never have negative damage:

void example(Base base) {
  print(base.damage); // 0, starting off undamaged.

  base.dealDamage(10);
  print(base.damage); // 10, after taking damage.

  base.healDamage(15); // Throws, would make damage negative.
}

Player

ResourceZone
A player's resource zone, where they store resources.

A zone shared by players that contains in-play ArenaCards.
SpaceArena
A shared zone representing the space arena.

A unit, which can be deployed to the ground or space arena.
Zone
