data property

List<CardOrVariant> data
latefinal

Cards (and variants) that belong to the expansion.

The cards are sorted by Card.number, such that for a full release (all cards in a single expansion), the card at index i has number i + 1:

final expansion = CatalogExpansion(...);
for (var i = 0; i < expansion.cards.length; i++) {
  print(expansion.data[i].card.number == i + 1); // true
}

WARNING: In practice, this is not always true, as some expansions will be partial releases, and some cards may be missing. Use the lookup method to reliably find cards by number:

final card = expansion.lookup(42);

The list is unmodifiable.

Implementation
